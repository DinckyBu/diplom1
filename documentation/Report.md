# Отчет по итогам тестирования
## Было проведено тестирование работы СУБД и API Банка.
### Этапы тестирования
1. Ручное тестирование.
2. Составление тест-плана.
3. Автоматизация тестирования.
4. Оценка результатов тестирования.
